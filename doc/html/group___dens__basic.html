◆ get_result()

double dmatrix::get_result ( mati  def,
qocircuit *  qoc 
)

Gets the probability of a counting event defined by def. In other words, we obtain the probability to find a specific number of photons, in a specific channel, polarization and with a determined frequency and time characteristics.

Parameters
matidef Is a 4xn matrix with the definition of the counting event of which the probability wants to be obtained where in each column:

1st-Row: Is the channels definition.
2nd-Row: Is the polarization definition.
3rd-Row: The wavepacket numbers. To obtain the probability of a counting event in a counter density matrix the packet index must be zero.
4th-Row: The number of photons (or counting events from a detector point of view).

Example:
in_term << 0, 1,
H, H,
0, 0,
3, 3;
prob=matrix->get_prob(in_term,example);
qocircuit*qoc Circuit where we want to perform detection.
Returns
It is returned the probability of the event defined by mati def.
See also
dmatrix *get_counter(int npack,qocircuit *qoc);
