◆ prnt_bins() [1/6]

void p_bin::prnt_bins ( )

Prints a set of probability bins.
Bin tags are printed in the same format that prnt_ket for the case where format = 0.

See also
prnt_ket(int iket, int format, bool loss,qocircuit *qoc);
Examples:
live1.cpp, and live2.cpp.
