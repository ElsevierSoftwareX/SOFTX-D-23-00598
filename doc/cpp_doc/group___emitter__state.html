)

Quantum dot state generator. It creates a state compatible with the photon emission of a quantum dot.

Parameters
matich Photon configuration matrix.
For each emitted photon we must define (in column) three numbers.
  • 1st Channel.
  • 2nd Wavepacket number if the photon is emitted in a state of horizontal polarization.
  • 3rd Wavepacket number if the photon is emitted in a state of vertical polarization.
doublek Fraction of the emitted photon pairs which originate from a XX-X cascade over a background of noise.
doubleS S/hbar Fine Structure Splitting (FSS) constant.
doubletx Exciton lifetime
doubletss Characteristic time of spin-scattering tss.
doublethv Characteristic time of cross-dephasing thv.
qocircuit*qoc Circuit to which the state to be created is related.
Returns
>=0 if success <0 otherwise.

Method to create a photonic Bell state in path encoding.

Parameters
matich Photon configuration matrix.
For each emitted photon we must define (in column) three numbers.
  • 1st Channel.
  • 2nd Wavepacket number if the photon is emitted in a state of horizontal polarization.
  • 3rd Wavepacket number if the photon is emitted in a state of vertical polarization. For each channel we must define (in column) two numbers. Channel and the corresponding wave function number for the photons emitted in that channel. There are as many columns as channels.
charkind Which of the four bell states is created.
'+'=|00> + |11>
'-'=|00> - |11>
'p'=|01> + |10>
'm'=|01> - |10>
doublephi Phase difference between the first and second ket in the definition of the Bell state.
| A, B > + exp(-i phi)| C, D >
qocircuit*qoc Circuit to which the state to be created is related.
Returns
>=0 if success <0 otherwise.

Method to create a photonic Bell state in polarization encoding.

Parameters
matich Photon configuration matrix.
For each channel we must define (in column) three numbers. Channel and the corresponding wave function numbers for the photons emitted in that channel. One packet number for H polarized photons and the other for P polarized photons. There are as many columns as channels.
charkind Which of the four bell states is created.
'+'=|HH> + |VV>
'-'=|HH> - |VV>
'p'=|HV> + |VH>
'm'=|HV> - |VH>
doublephi Phase difference between the first and second ket in the definition of the Bell state.
| A, B > + exp(-i phi)| C, D >
qocircuit*qoc Circuit to which the state to be created is related.
Returns
>=0 if success <0 otherwise.

Quantum dot photon pair generator. It creates a photon pair as it would be generated by a quantum dot excited by a single laser pulse.

Parameters
inti_ch0 Channel were the first photon is created.
inti_ch1 Channel were the second photon is created.
vecii_t Vector with the wavepacket numbers to be assigned to the photon depending on their polarization. First, the first channel horizontal and vertical wavepackets and then the second channel in the same order 1H1V2H2V.
doublek Fraction of the emitted photon pairs which originate from a XX-X cascade over a background of noise.
doubleS S/hbar Fine Structure Splitting (FSS) constant.
doubletx Exciton lifetime
doubletss Characteristic time of spin-scattering tss.
doublethv Characteristic time of cross-dephasing thv.
qocircuit*qoc Circuit to which the state to be created is related.
Returns
>=0 if success <0 otherwise.
