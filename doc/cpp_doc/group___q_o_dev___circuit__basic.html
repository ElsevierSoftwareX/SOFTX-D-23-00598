◆ delay()

int qodev::delay ( int  ch)

Increases the optical path of a channel by a quantity equal to one period.
Furthermore it also adds the frequency dependent phase caused by the delay.

Parameters
inti_ch Channel where the delay is introduced.
Returns
0 if success -1 if error.
KnowIss:
Delay can not be called if the circuit is configured to calculate explicit losses (flag loss=true in qocircuit constructor).
