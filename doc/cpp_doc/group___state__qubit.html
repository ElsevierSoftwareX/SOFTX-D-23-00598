Detailed Description

Support to encode and decode qubit states into photonic states (and back) using path encoding.

Function Documentation

◆ encode()

state * state::encode ( mati  qbits,
qocircuit *  qoc 
)

It encodes a photonic state into a qubit state (path encoding).
Only for ideal circuits. nm=1 and ns=1

Parameters
matiqbits Qubit definition. Matrix with a column entry for each qubit where they are defined the two photonic channels needed to encode the qubit. A 01 occupation means Q=0 and a 10 occupation Q=1.
qocircuit*qoc Circuit to which the state is related.
Returns
Encoded state.
