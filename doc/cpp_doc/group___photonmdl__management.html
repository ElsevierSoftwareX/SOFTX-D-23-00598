photon_mdl::photon_mdl ( mati  pack_def,
vecd  times,
matd  freq,
char  ckind 
)

Creates a packet model given a set of parameters. These are, a packet definition and lists of available times and frequencies.

Parameters
matipack_def Packet definition. It is an integer matrix of three rows where each column contains, a wave packet.
1st row = an index that labels the packet uniquely.
2nd row = an index that refers to the central time of the packet as specified in the matrix times.
3rd row = an index that refers to the central frequency and width of the packet as specified in the matrix freq.
vecdtimes Vector that contains the discrete sets of times in which a packet may exist in the current simulation.
matdfreq Two row matrix that contains the discrete set of frequency configurations that a packet may have in the current simulation.
1st row = is the packet central frequency.
2nd row = is the packet width or characteristic time.
charckind Kind of the wave packet 'G'=Gaussian, 'E'=Exponential.
